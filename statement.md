## Problem Statement

Bioengineering students need an easy way to self-assess and reinforce learning with immediate feedback through interactive quizzes.

## Scope of the Project

A Google Colab app that enables user registration, login, and interactive quizzes on bioengineering topics, with instant scoring and feedback.

## Target Users

Bioengineering students and educators seeking quick, interactive quiz-based assessment in a notebook environment.

## High-Level Features

- User registration/login
- Multiple bioengineering questions
- Interactive quiz interface
- Instant scoring and feedback
- Runs natively in Google Colab with no setup needed
